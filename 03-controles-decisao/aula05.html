Aula 05 - switch.. case

Vamos resolver um problema?

Pedir para o usuário digitar 1 número de 0 à 6. Mostrar uma mensagem informando qual é o dia da semana correspondente. Domingo (0) até Sábado (6).
Exemplo: se o usuário digitar 3, deve mostrar "Quarta-feira."
