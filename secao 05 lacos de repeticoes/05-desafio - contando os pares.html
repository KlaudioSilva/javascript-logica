Pedir ao usuário para digitar um número qualquer e mostrar
todos os números pares do 2 (inclusive) até o número que o usuário digitou.

PARA NÃO ESQUECER

  1. Um loop SEMPRE precisa ter uma condição de saída
  2. Loop For: ideal para quando sabemos exatamente quantas vezes o loop será executado
  3. Loop While: ideal para quando não sabemos quantas vezes o loop será executado. Além disso, se a condição for false, não executa nenhuma vez
  4. Loop Do..While: se parece com o loop While, mas o código é executado ao menos uma vez, mesmo que a condição seja inicialmente false.
