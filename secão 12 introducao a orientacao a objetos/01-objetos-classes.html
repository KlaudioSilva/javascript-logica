Overview Sobre Orientação à Objetos

Paradigma de programação baseada na ideia de "objetos" do mundo real e suas interações.

O que é um Objeto?
Um objeto é uma unidade que pode descrever suas caracteristicas (propriedades, atributos) ou suas funções (métodos).
Esse Objeto é coeso, pois agrupa tanto os dados quanto sua funções.

Um objeto nesse paradigma precisa ser coeso, ou seja, precisa agrupar sua propriedades e métodos (estes podem modificar sua propriedades).

No trecho de código abaixo, o que acontece se tivermos que modificar a lógica do método 'comprar'? E se tivermos 10, 20, 100 produtos diferentes?

const caneta = {
    nome: 'Caneta',
    quantidade: 10,
    comprar: function (qtd){
        this.quantidade -= qtd
    }
}

const caderno = {
    nome: 'Caderno',
    quantidade: 7,
    comprar: function (qtd){
        this.quantidade -= qtd
    }
}

Como criamos Objetos em POO?
Em linguagens orientadas à objetos, um objeto pode ser criado à partir de um molde (projeto) que, em POO, chamamos de classe.

O que é uma classe?
Em linguagens orientadas à objetos, uma classe é uma especificação (um molde, um projeto). À partir dessas classes criamos objetos.
NUNCA usamos classes. SEMPRE usamos objetos.

À partir de uma classe (projeto) criamos vários objetos iguais.
Se modificarmos a classe, estaremos modificando também todos os objetos gerados à partir dela. Mas se modificarmos apenas um objeto nada muda na classe ou nos outros objetos.

Em POO dizemos que cada objeto gerado por uma classe é chamado de Instância daquela determinada classe.

Mesmo que nos é permitido trabalhar com objetos literais, podemos deixar nosso código otimizado para futuras manutenções ao utilizar classe e funções construtoras.

Uma função construtora é um método especial para inicializar um objeto criado à partir de uma classe.

class Produto{
    constructor (nome, qtd){
        this.nome = nome
        this.quantidade = qtd
    }
    comprar(qtd){
        this.quantidade = qtd
    }
}

const caneta = new Produto('Caneta', 10)
const caderno = new Produto('Caderno', 16)
